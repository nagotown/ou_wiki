## What is VOICEVOX?
[VOICEVOX](https://voicevox.hiroshiba.jp/) is a free, medium-quality text-to-speech and singing voice synthesis software.

## How to install
<img width="45%" height="45%" alt="image" src="https://github.com/user-attachments/assets/377f368d-fda8-45ef-9678-88479da8adea" />  

Enable Install to Additional Singer Path.  
DL [VOICEVOX](https://voicevox.hiroshiba.jp/) with the whole editor.  
DL the [VOICEVOX_Singer.zip](https://github.com/rokujyushi/OpenUtau/releases/tag/999.999.999.999.999) and 
D&D it into the main window of OpenUtau.  
Start OpenUtau while running VOICEVOX.

An engine-only dependency package will be created in the future. (This is just a plan)

## Phonemizers
The following are currently supported.  
* S-VOICEVOX JA (Simple Voicevox Japanese Phonemizer)  
* S-VOICEVOX EN to JA (Simple Voicevox ENtoJA Phonemizer)
* VOICEVOX JA (Voicevox Japanese Phonemizer)  
 Supports editing of phoneme length.

## Expressions
The following are currently supported.
* DYN (Dynamics)
 Allows volume adjustment after speech synthesis
* SHFT(tone shift) 音域調整
 For adjusting the input pitch of speech synthesis (in phrases).
 It changes the way you sing.
* CLR(Voice Color)
 Changes the voice quality of each singer (per phrase).
* VOL(Volume) 声量調整
 Adjustment of voice synthesizer input volume (per phrase)
* VOLC(input volume (curve)) 声量調整
 Adjusting the voice synthesizer input volume (per frame)
* PITD(pitch curve)
 Adjusts the text-to-speech input pitch (in frames).
* DIR (direct)
 Mode switching to synthesize based on the pitch generated by the VOICEVOX engine
* SMOC(pitch smoothened (curve))
 Smoothing of pitches loaded with LoadRenderedPitch
* DUCM(duration correction mode)
 VOICEVOX JA timing correction function on/off switch (default auto)

To be supported in the future
* SHFC (tone shift curve)
* BSNG (bass singer)
